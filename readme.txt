Install VSCode extensions:
Extension Pack For Java
Community Server Connectors
Once installed, open the Servers tab (should be below your file list on the left pane)
Right click on Community Server Connector
Click "Create New Server"
Choose to download a new server, and choose Glassfish 7
Once installed, right click on the new glassfish7 item and click "Start Server"
After running the server, press Ctrl+Shift+B to compile and package the project
Then right-click on the new "iotbay.war" file and click "Run in server"
The website will be available in your browser at http://localhost:8080/iotbay
The website *should* also update content if you change something by just pressing Ctrl+Shift+B again
